Statistiques

Raison d’être de ces jeux de données

Ces jeux de données sont le résultat de l'agrégation aux mailles administratives d'indicateurs, comme le nombre d'éligibles à tel(le) ou tel(le) technologie/débit, les inéligibles, etc.

Données sources

Ces données sont produites à partir :

  • de l'éligibilité ;
  • des fichiers des opérateurs d'infrastructures ;
  • des contours administratifs Admin Express de l'IGN;

Processus de construction

Les données d'éligibilité sont croisées avec les contours administratifs.

Couverture

L'ensemble du territoire métropolitain est couvert ainsi que les Départements d'Outre-Mer.

Format(s) disponible(s)

Les données sont actuellement disponibles au format csv.

Mise à disposition

Les données sont mises à disposition en téléchargement direct à la région, au département, à la commune ou France entière.
Ce sont ces URL qu’il faut utiliser si vous souhaitez automatiser la récupération des données et bénéficier des meilleures performances.

Historique

  • 08/04/2021 : première diffusion des données

Millésimes disponibles en téléchargement direct

Les liens suivants permettent de télécharger les données par région, par département, par commune, ou France entière.

NB : Il est possible de remplacer la date du millésime par `last` dans les URL.

Données de référence - Technologies

Nom Code statistique Code Description
DSL log_cu CU Accès fixe à Internet via les lignes cuivre du réseau téléphonique.
Câble log_coax COAX Accès fixe à Internet via le réseau coaxial de télévision câblée.
Fibre log_ftth FO Accès fixe à Internet via un réseau fibres optiques.
4G Fixe log_4gf 4GF Accès fixe à Internet via un routeur 4G. Les capacités du réseau sont partagées entre les usages fixes et mobiles.
THD Radio log_thdr THDR Accès fixe à Internet via un réseau radio à très haut débit. Les équipements utilisent les mêmes technologies que les réseaux mobiles 4G mais le réseau est dédié à cet usage.
HD Radio log_hdr WMX Accès fixe à Internet via un réseau radio à haut débit.
Satellite log_sat SAT Accès fixe à Internet via des faisceaux satellitaires pour des territoires particulièrement mal connectés.
